# Gocomet Automation
## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Gocomet_Automation
```

2. Install dependencies:
```bash
npm install
```

3. Install Playwright browsers:
```bash
npx playwright install
```
4. To execute all the test case:
```bash
npm run all
```
5. To execute smoke test case:
```bash
npm run smoke
```

6. To execute the test cases in ui mode
```bash
npx playwright test --ui
```
